20 Seconds To Launch

20 Seconds To Launch est un jeu 2D réalisé en équipe sur 48 heures lors de la Global Game Jam 2020. Le thème de cette game jam été "réparer".

Le jeu consiste à réparer une fusée avant son lancement, dans 20 secondes. Il se base sur le concept du die and retry, c'est-à-dire que le joueur doit essayer encore et encore jusqu'à trouver les bonnes actions à faire décoller la fusée.

  • Langages: Unity3D (C#)
  • Repository: tguichaoua/20-seconds-to-launch
Game Jam

Source: Wikipédia

Une game jam est un hackathon avec pour thème principal les jeux vidéo, mais pouvant aussi porter sur les jeux au sens large. Les participants, groupés en équipes, doivent créer un jeu dans un temps limité, généralement le temps d'un week-end, sur 24 à 72 heures.
